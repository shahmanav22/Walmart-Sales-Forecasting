# Walmart Sales Forecasting using Machine Learning

## Overview

This project predicts Walmart weekly sales using historical sales data and machine learning regression models. It demonstrates an end-to-end forecasting workflow, including data preprocessing, exploratory data analysis, feature engineering, model training, evaluation, and business recommendations.

## Objectives

- Forecast future weekly sales
- Analyze factors affecting sales
- Compare multiple regression models
- Generate actionable business insights

## Dataset

- Walmart Sales Forecasting Dataset (Kaggle)

## Workflow

- Data Cleaning
- Exploratory Data Analysis
- Statistical Analysis
- Feature Engineering
- Time-Based Train-Test Split
- Regression Modeling
- Model Evaluation
- Sales Forecasting
- Business Recommendations

## Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

## Evaluation Metrics

- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SciPy
- Joblib

## Key Business Insights

- Holiday periods significantly influence sales.
- Seasonal trends affect customer demand.
- Historical sales are strong predictors of future sales.
- Inventory planning can be optimized using forecasting.
